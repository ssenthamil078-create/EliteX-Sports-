# AI Sports Frontend — Phase 1

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

## Backend URL

Create `.env` from `.env.example`:

```bash
VITE_API_BASE_URL=http://127.0.0.1:8000
```

## Phase 1 Includes

- React + Vite
- TailwindCSS sports theme
- 3D/glassmorphism UI foundation
- Auth store with Zustand
- Axios backend service
- React Query setup
- Toast notifications
- Protected routes
- Sidebar and topbar
- Login/Register/Forgot Password
- Dashboard shell
- Legal pages from backend
